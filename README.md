## DateTime

#### Get Household date format easily

    DateTime dateTime = DateTime(2019, 2, 3);
    dateTime.toStringYYYYMMDD("-")  // "2019-02-03"
    dateTime.toStringYYYYMMDD("/")  // "2019/02/03"
